Apps and functions

This section contains the full documentation of all apps (GUIs) and analysis functions of the umIToolbox.

Apps & tools

Apps
umIToolbox the main user interface
DataViewer use to visualize imaging recordings (works as standalone)
ROImanager creates and manages of Regions of Interest (ROIs) (works as standalone)

Tools
Pipeline Configuration creates and edits analysis pipelines (from DataViewer and umIToolbox apps)
Manual Image Registration aligns imaging data using control point selection (from umIToolbox app)
Line/Box plotter plots grouped longitudinal data (from umIToolbox app)
Correlation matrix plotter plots correlation matrices (from umIToolbox app)
Multi-plot plots up to 3 extra recordings alongside the currently loaded dataset (from DataViewer app)
