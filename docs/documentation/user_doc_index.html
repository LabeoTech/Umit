Apps and functions

This section contains the documentation of all apps (GUIs) and analysis functions of the umIToolbox.

Apps & tools

Apps
umIToolbox the main user interface
DataViewer use to visualize imaging recordings (works as standalone)
ROImanager creates and manages of Regions of Interest (ROIs) (works as standalone)

Tools
Multi-plot plots up to 3 extra recordings alongside the currently loaded dataset (from DataViewer app)
Merge recordings Concatenates recordings into a single .dat file (from umIToolbox app)
Pipeline Configuration creates and edits analysis pipelines (from DataViewer and umIToolbox apps)
Manual Image Registration aligns imaging data using control point selection (from umIToolbox app)
Line/Box plot plots grouped longitudinal data (from umIToolbox app)
Correlation matrix plotter plots correlation matrices (from umIToolbox app)
Plot Average Maps Plots group average of activation maps (from umIToolbox app)

Preprocessing and analysis functions

Here is a list of all analysis functions currently available in umIT separated by category:

Data Import
importFromTif imports imaging data stored in multi-frame .TIF files
run_ImagesClassification imports recordings acquired with LabeoTech's Imaging systems

Data Export
run_ConvertToTiff exports imaging data to multi-frame .TIF file(s)

Filters
GSR performs Global signal regression in image time series data
apply_aggregate_function applies an aggregate function (e.g. mean) to a dimension of data
spatialGaussFilt Applies smoothing to image time series using a Gaussian filter
normalizeBSLN Normalizes the data by baseline
normalizeLPF Normalizes image time series data using low-pass filtering
normalizeZScore Normalizes image time series data as z-score in time domain

Imaging
SPCM Creates seed-pixel correlation maps from image time series
alignFrames Performs automatic image registration to image time series
apply_detrend Removes linear trend from the time domain of image time series
calculate_response_amplitude Creates response amplitude maps for image time series split by events
run_Ana_Speckle Calculates the blood flow from the Laser Contrast Speckle Imaging (LSCI) data
run_HemoCompute Calculates HbO and HbR from reflectance signals
run_HemoCorrection Performs hemodynamic correction of the fluorescence signal
run_SpeckleMapping Calculates the average speckle contrast map from Laser speckle raw data
split_data_by_event Rearranges event-triggered data by trials
trim_movie Crops the start and/or end of an image time series

Retinotopy
genRetinotopyMaps Creates azimuth and elevation maps from retinotopic mapping recordings.
genVSM Creates a visual sign map using the azimuth and elevation maps.

Analysis functions
getDataFromROI Extracts and aggregates ROI data from imaging datasets
genCorrelationMatrix Creates a correlation matrix from ROIs of image time series
transformDat2Mat Saves imaging data from .dat to a .mat file compatible with umIToolbox Analysis tab.
