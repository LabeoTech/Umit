Change observation/event lists - tool

This tool allows one to reorder and subset observations and/or events.

How to change a list

In the following example, we will reorder the observation list and subset the event list. The data contains a total of 25 observations and 25 events:

Reordering observations

To reorder the observations:

  1. Click on Change Lists button
  2. Select observation list
  3. In the app, select one item from the New list and use the up/down arrows to reorder
  4. Click on Confirm to apply the changes

Subsetting events

To subset a list:

  1. Click on Change Lists button
  2. Select event list
  3. In the app, click on the Reset button
  4. Select one or more items from the Original list
  5. Click on the >>Add>> button
  6. Click on Confirm to apply the changes

Resetting list

To reset a list to the original items and order:

  1. Click on Change Lists button
  2. Select the desired list
  3. In the app, click on the Reset button
  4. Select all items from the Original list
  5. Click on the >>Add>> button
  6. Click on Confirm to apply the changes

Save and load lists

The edited lists can be saved to a .mat file and loaded for later use with the same dataset or a different one.

Note
If a different dataset is used, be sure that all the items from the saved list are contained in the original list of the selected data.
