transformDat2Mat

Saves imaging data from .dat to a .mat file compatible with umIToolbox Analysis tab.

Description

This function transforms an imaging data file from .dat to a .mat file to be used as input in the Analysis tab of the umIToolbox app.

Input

This function accepts any imaging data from a .dat file.

Output

The output of this function is a .mat file containing the input data linked to a generic ROI with name genericObs.
