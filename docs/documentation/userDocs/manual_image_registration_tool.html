Manual image registration tool

The manual image registration tool is available from the Visualization tab of the umIToolbox app. This tool uses landmarks to align a recording to a reference frame. The landmark selection is performed interactively using Matlab's control-point selection tool cpselect. Then, the geometric transformations used to fit the points to the reference frame is applied to the target imaging file. For instructions on how to create a reference frame, read the documentation of the ROImanager app.
