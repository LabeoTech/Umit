alignFrames - Image registration

Performs automatic registration of imaging data to a reference frame.

Description

This function performs the automatic alignment of an image dataset to an Imaging Reference frame. The creation of the reference frame is performed by the ROImanager app and is described here. The function uses one imaging data (stored in a .dat file) to align with the reference frame and applies the geometric transformation to the input file.

Input

This function accepts only images (with dimensions Y,X) or image time series (with dimensions Y,X,T) as input.
