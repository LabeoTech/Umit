Visualize grouped data

Until now, you learned how to use umIT to import, preprocess, extract and aggregate data from imaging datasets. Now, the data from individual recordings need to be put together in their respective experimental groups and visualized. This is an important step to perform the exploratory and inferential data analysis necessary to answer your scientific questions.

Here, we will use the correlation matrices generated in our previous tutorial on how to extract ROI data as an example to illustrate the data grouping and visualization. The toolbox provides a tool dedicated to the visualization of correlation matrices called Plot Correlation Matrix. This and other tools are available in the Apps & tools panel in the Analysis tab of the main GUI (umIToolbox). To learn about the other tools, please read the documentation of the umIToolbox app.
