Getting started

Toolbox to manage, process and visualize widefield imaging experiments.

The umIToolbox is dedicated to the preprocessing, visualization and analysis of mesoscale imaging datasets. umIT provides tools to assist on experiment and data management, automation of data processing pipelines as well as imaging data visualization.

Although the toolbox was created to work with large scale datasets, you can also use our DataViewer and ROImanager apps as standalone to quickly pre-process, visualize and extract data from single imaging recordings.
